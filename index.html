📝 De un Vistazo: Comando Central

Todo lo que necesitas para preparar y ejecutar esta misión en minutos.

🎯 Objetivo de Aprendizaje

Los estudiantes podrán: Comprender la coordinación de sistemas y la estabilidad de ejecución, y usar el Micro:Bit y el Kit Nezha Pro para integrar un servo en una secuencia de movimiento con timing preciso.

🤖 Prompt de Referencia para XOLO (Solo para Maestro)

⚠️ NOTA: XOLO ya conoce el tema de esta misión. Este prompt es SOLO de referencia en caso de emergencia técnica: 
Ayúdame a integrar el servo al vehículo y a ejecutar la secuencia completa: mover → detener → activar servo → mover. Guíame para probar si la acción ocurre a tiempo, si el servo se activa demasiado pronto o tarde, y cómo ajustar pausas y duración para que la secuencia se vea coordinada.

⚡ Puntos Esenciales de la Misión

🎓

Concepto Clave

Coordinación de sistemas: Es como una coreografía donde diferentes partes del robot deben trabajar juntas en el momento exacto para que todo funcione bien.

🔧

Habilidad Técnica

Usar un motor servo para realizar una acción específica dentro de una secuencia de movimiento.

🧪

Experimentación

Probar con diferentes tiempos de pausa y ángulos del servo para entender limitaciones de estabilidad.

💬

Interacción con XOLO

XOLO ya conoce a cada estudiante y el tema de hoy. Solo abren el chat y comienzan a aprender. Tú guías el timing y motivación.

🚨 Resolución Rápida de Problemas

  • Si los alumnos no pueden ingresar a Gnius Space: Activa XOLO SOS en xolo.gnius.club para continuar la misión sin interrumpir la clase.
  • Si el servo no se mueve: Verificar que esté conectado en el puerto correcto (S1, S2, etc.) del Kit Nezha y que el cable no esté al revés.
  • Si el robot se mueve de forma errática: Revisar que los motores estén bien conectados y que la batería tenga suficiente carga para alimentar tanto motores como el servo.
  • Si hay errores en el código: Recordar a los estudiantes que XOLO puede ayudar a depurar. Pídeles que copien el error y se lo muestren a XOLO.

📊 Diapositiva 3: Brain Spark

1. 💬 Pide a XOLO que te ayude a crear:

"Xolo, ayúdame a idear mejoras para que la acción combinada sea más precisa y realista usando Python. Mi vibe: ‘robot con misión clara y movimientos coordinados’. Dame 2–3 mejoras y conviértelo en GCS + código Python según tu entrenamiento con el robot XGO y Micro:bit."

2. 🔬 Experimenta y descubre con tu programa:

  • ¿El robot realiza la secuencia completa (mover, parar, servo, mover) sin trabarse?
  • ¿Qué pasa si el servo intenta moverse mientras el robot todavía está avanzando?
  • ¿Funciona igual de bien la secuencia si aumentas la velocidad de los motores?
  • ¿Cómo ayuda el tiempo de espera (pausa) a que los diferentes sistemas del robot estén coordinados?
  • ¿Por qué es importante que el robot se detenga por completo antes de activar el servo para mantener la estabilidad?

¡Es hora de crear algo asombroso! Vamos a idear, definir un VIBE, obtener nuestro GCS y luego programar en MakeCode para experimentar. Por ejemplo, podrían crear un robot recolector que avanza hacia un objeto, se detiene, usa su brazo servo para recogerlo y luego sigue su camino. Es fundamental que sigan las preguntas de experimentación para ver si su robot es estable o si se tambalea. Cuando tengan su código listo en el simulador, levanten la mano para que les entregue su hardware. Si tienen errores o dudas, pregúntenle a XOLO, él los ayudará a depurar su código para que la misión sea un éxito total.
